# ⚡ Peer Power

Peer-to-peer EV charging platform.

## 🚗 Problem
EV drivers often struggle to find available chargers, especially in residential areas.

## 💡 Solution
Peer Power allows users to:
- Find nearby private chargers
- Book them instantly
- Pay directly in the app

Hosts can monetize their unused chargers.

## 💸 Business Model
We take a commission from each booking.

## 📱 MVP Features
- Charger listings
- Booking system
- In-app balance & payments
- User profile

## 🚀 Status
Working MVP built with React Native (Expo)
